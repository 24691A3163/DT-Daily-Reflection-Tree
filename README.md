# DT Daily Reflection Tree

This project implements a deterministic reflection tool.

## Overview
The system guides users through a structured reflection based on three axes:

1. Locus of Control (Victim vs Victor)
2. Contribution vs Entitlement
3. Radius of Concern (Self vs Others)

## How It Works
- Users answer fixed-choice questions
- Each answer leads to a predefined path
- The system generates consistent reflections

## Key Features
- Fully deterministic (no randomness)
- No LLM used at runtime
- Structured decision tree logic

## Structure
- tree/ → reflection tree and diagram
- write-up.md → design explanation

## Goal
To help users reflect on their behavior in a structured and meaningful way.
